# odbc

A new experimental ODBC wrapper.

## Status

Very much alpha software. I expect breaking changes. You should feel hesitant using it in your projects.

## Installation

* Install [corral](https://github.com/contact-red/odbc)
* `corral add github.com/contact-red/odbc.git --version 0.0.3`
* `corral fetch` to fetch your dependencies
* `use "odbc"` to include this package
* `corral run -- ponyc` to compile your application
